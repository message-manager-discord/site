Message Manager

Invite Documentation

About

Message Manager is designed to make the management of important messages in servers easier.
It allows the editing on a single message by more than one staff member.

Features

  • Message management (editing and sending)
  • Customizable prefix and server settings

Example Usage

Setup

  1. Invite the bot to your server
  2. Setup the prefix with ~setup prefix
  3. Then setup the rest of the configuration.

Contributing

This bot is open source, the bots code can be found at message-manager-discord/bot. And the website's code can be found at AnotherCat/message-manager-site
Issues on github and the support server can be used to raise issues and or feature requests on the relevant repository.
If you want to contribute to the bots code please open a pull request.
